## 二、配色体系

### 2.1 主色（品牌色）

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#ff6b9d` | 粉红，用于按钮、选中态、强调 |
| 主色深 | `#c44569` | 深粉/玫红，渐变终点 |
| 主色浅 | `#ffa6c1` / `#F18EB2` | 浅粉，标题区、背景 |

| 场景 | 渐变值 | 使用位置 |
|------|--------|----------|

| 标题区浅 | `linear-gradient(135deg, #ffa6c1 0%, #ff6b9d 100%)` | 分析工作台标题 |
| 背景浅 | `linear-gradient(135deg, rgba(255,166,193,0.25) 0%, rgba(255,107,157,0.12) 50%, rgba(196,69,105,0.08) 100%)` | Lottie 动画区 |

| 完成态 | `linear-gradient(135deg, #d4edda 0%, #c3e6cb 100%)` | 采集完成卡片 |

## 三、布局与组件结构

### 3.1 全局布局

- **Header**：64px 高，主渐变背景，左侧 Logo + 标题 + 部门标签，右侧用户头像下拉
- **Sider**：240px 宽，白色背景，根据部门显示不同菜单项
- **Content**：`#f5f6fa` 背景，32px 内边距

### 3.2 页面通用结构

多数页面采用：

1. **页面标题区**：主渐变背景，大标题 + 副标题
2. **内容卡片**：白色 Card，16px 圆角，无边框，`box-shadow: 0 4px 16px rgba(0,0,0,0.08)`

### 3.3 企业社群布局

- 三栏：左侧部门导航（240px）→ 中间帖子列表（360px）→ 右侧帖子详情
- 部门/小群选中态：使用群配置的 `color` 或默认 `#ff6b9d` / `#a29bfe`

## 四、字体与间距

- **字体**：Inter + 系统字体（PingFang SC、Microsoft YaHei 等）
- **标题**：36px / 24px，font-weight 700
- **正文**：15px / 14px，font-weight 500
- **表单**：表单项 label 16px/600，输入框 15px
- **圆角**：卡片 16px，按钮 12px / 24px / 28px（胶囊），Tag 12px
- **阴影**：`0 4px 16px rgba(0,0,0,0.08)`、`0 8px 24px rgba(255,107,157,0.2)` 等

## 五、UI 优化建议

### 5.1 配色与视觉

| 问题 | 建议 |
|------|------|
| 主色单一 | 可增加主色深浅层级，用于不同交互态（hover/active/disabled） |
| 渐变使用过多 | 部分区域可改为纯色，减少视觉噪音 |
| 中性灰不统一 | 统一 `#2d3436` / `#636e72` / `#95a5a6` 的使用场景 |
| 占位页单调 | 色号设计、虚拟试妆、内容生成可增加插画或示意 UI |

### 5.2 布局与响应式

| 问题 | 建议 |
|------|------|
| 企业社群三栏固定宽度 | 小屏下改为可折叠或单栏流式布局 |
| 分析工作台表单较长 | 可折叠「高级选项」减少首屏高度 |
| 报告页图表多 | 考虑懒加载或分 Tab 展示 |

### 5.3 交互与反馈

| 问题 | 建议 |
|------|------|
| 爬虫进度为 Mock | 若后端支持真实进度，可替换为真实进度 |
| 分析历史无筛选 | 增加按平台、时间筛选 |
| 数据列表导出仅 CSV | 可增加 Excel 格式导出 |
| 社群发帖无草稿 | 可增加本地草稿保存 |

### 5.4 可访问性与性能

| 问题 | 建议 |
|------|------|
| 部分图标无 aria-label | 为图标按钮补充无障碍说明 |
| 大列表无虚拟滚动 | 数据列表、社群帖子列表可考虑虚拟滚动 |
| 图表无降级 | 无数据时提供占位图或友好提示 |

### 5.5 设计系统建议

1. **抽离设计 Token**：将主色、中性色、圆角、阴影等写入 CSS 变量或主题文件，便于统一调整
2. **组件库定制**：基于 Ant Design 做主题定制（`ConfigProvider`），减少内联样式
3. **暗色模式**：如需暗色模式，可提前规划 Token 结构
